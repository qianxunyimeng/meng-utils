deepMergeKey

  • deepMergeKey(original: unknown, arrayProcessMethod: boolean, ...objects: any[]): any

  • 深度合并对象，替换叶子结点同名属性 不会合并原型链上的属性

    example
    const config = {
        userName: {
          first: 'A',
          second: 'B',
        },
        age: 2,
       hobby:["读书"]
     };
     const config2 = {
      userName: {
         first: "C"
        },
        hobby:["唱歌"]
      }
    console.log(JSON.stringify(mu.deepMergeKey(config,false,config2))); // -> {"userName":{"first":"C","second":"B"},"age":2,"hobby":["读书","唱歌"]}

    Parameters

    • original: unknown

      原始对象

    • arrayProcessMethod: boolean

      数组处理方式

      • true 表示替换新值，不管新值为哪种类型
      • false 表示会合并整个数组（将旧数据与新数据合并成新数组）
    • Rest ...objects: any[]

      要合并到对象

    Returns any

extend

  • extend(target: any, deep?: boolean, ...objs: any[]): any

  • 将多个对象属性复制到目标对象身上。 如果deep=true extend将执行深度复制，递归复制它找到的任何对象。否则，副本将与原始对象共享结构。 未定义的属性不会被复制,但是，从对象原型继承的属性将被复制

    Parameters

    • target: any

      目标对象

    • deep: boolean = true

      是否执行深度复制

    • Rest ...objs: any[]

    Returns any
